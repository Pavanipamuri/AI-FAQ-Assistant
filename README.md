# AI FAQ Assistant

A simple AI-powered FAQ chatbot built using Python, NLTK, and Scikit-learn.
The chatbot answers predefined frequently asked questions using Natural Language Processing techniques.

## Features

* AI-based FAQ responses
* Natural Language Processing using NLTK
* Text vectorization using Scikit-learn
* Simple command-line chatbot
* Beginner-friendly project structure

## Technologies Used

* Python
* NLTK
* Scikit-learn
* NumPy

## Project Structure

```text
AI-FAQ-Assistant/
│── chatbot.py
│── requirements.txt
│── README.md
```

## Installation

Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python chatbot.py
```

## Example Questions

* What is AI?
* What is Python?
* What is Machine Learning?

## Output Example

```text
FAQ Chatbot
Type 'exit' to quit

You: What is AI?
Chatbot: AI stands for Artificial Intelligence.
```
